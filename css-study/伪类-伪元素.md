# 伪类

CSS 伪类是添加到选择器的关键字，指定选择器的特殊状态。
伪类的特征是冒号(:)

```css

button:hover {
  color: blue;
}

```

# 伪元素
CSS 伪元素是附加到选择器的关键字，允许你的选择器
伪元素的特征是两个冒号(::)
```css

/* 每一个 <p> 元素的第一行。 */
p::first-line {
  color: blue;
  text-transform: uppercase;
}

```
